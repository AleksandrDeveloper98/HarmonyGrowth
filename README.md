## 📋 О проекте

Современный адаптивный лендинг с плавными якорными ссылками. Разработан для центра помощи детям с особенностями развития.

### 🛠 Технологии

- **Minista** — статический генератор
- **TypeScript** — типобезопасность
- **SASS/SCSS** — модульные стили
- **Адаптивный дизайн** — мобильная верстка

### ✨ Особенности

- ✅ Плавный скролл по якорным ссылкам
- ✅ Адаптивная верстка (desktop → tablet → mobile)
- ✅ Бургер-меню с анимацией
- ✅ Градиентные кнопки и иконки
- ✅ Форма обратной связи
- ✅ Семантическая верстка
